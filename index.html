Introduction

THIS IS A DEV PREVIEW

Jade Application Kit(JAK) is an application build in Python 3, on top of a Webkit2 Webview using Pygi and GTK as a container to display HTML5 contents its own native desktop window. Allowing You to write hybrid Web and Desktop applications on Linux.

This works same way as Node Webkit or Electron, If you want something fully cross platform you can use one of them instead.
If linux is all you want you can give this a try!

Use any scripting language you like ( PHP, Python, Ruby, Javascript )!

Use HTML5, CSS3, Webgl.

Use any backend server or none at all!

Have your applications run in the browser with websockets using the GTK Broadway backend.
